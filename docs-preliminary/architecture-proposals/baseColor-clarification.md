# Уточнение: baseColor vs dark.color

**Дата**: 2025-11-05  
**Вопрос**: В чём разница между `baseColorDark` и `dark.color`?

---

## Текущая структура

```json
{
  "segments": {
    "architecture": {
      "baseColorDark": "#1976D2",    // ← базовый цвет сегмента
      "baseColorLight": "#E3F2FD",
      "types": {
        "concept": {
          "dark": {
            "color": "#1976D2",      // ← цвет заливки узла
            "border": "#0D47A1"      // ← цвет обводки
          }
        }
      }
    }
  }
}
```

---

## Проблема

Оба значения (`baseColorDark` и `dark.color`) — это цвета заливки. В примере они совпадают (`#1976D2`), что создаёт дублирование.

---

## Варианты решения

### Вариант 1: Убрать baseColor (если не используется)

Если `baseColor` не используется в коде, его можно убрать:

```json
{
  "segments": {
    "architecture": {
      "types": {
        "concept": {
          "dark": {
            "color": "#1976D2",  // цвет заливки
            "border": "#0D47A1"  // цвет обводки
          }
        }
      }
    }
  }
}
```

### Вариант 2: Использовать baseColor для UI элементов

Если `baseColor` нужен для визуального обозначения сегмента в UI (легенда, фильтры, группировка), то:

```json
{
  "segments": {
    "architecture": {
      "baseColorDark": "#1976D2",   // для легенды/фильтров
      "baseColorLight": "#E3F2FD",
      "types": {
        "concept": {
          "dark": {
            "color": "#1976D2",     // для заливки узла (может отличаться)
            "border": "#0D47A1"
          }
        }
      }
    }
  }
}
```

**Использование**:
- `baseColor` → для легенды, фильтров, группировки узлов по сегментам
- `dark.color` → для заливки конкретного узла

### Вариант 3: baseColor как средний цвет сегмента

Если типы узлов в сегменте имеют разные цвета, `baseColor` может быть средним/основным цветом сегмента:

```json
{
  "segments": {
    "architecture": {
      "baseColorDark": "#1976D2",   // средний цвет сегмента
      "baseColorLight": "#E3F2FD",
      "types": {
        "concept": {
          "dark": { "color": "#1976D2" }  // синий
        },
        "technology": {
          "dark": { "color": "#388E3C" }  // зелёный
        },
        "version": {
          "dark": { "color": "#7B1FA2" }  // фиолетовый
        }
      }
    }
  }
}
```

---

## Рекомендация

**Убрать `baseColor`**, если он не используется в коде.

Если нужен для UI, то:
- Использовать для легенды/фильтров
- Или убрать и брать цвет из первого типа узла сегмента
